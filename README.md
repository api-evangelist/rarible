# Rarible

Rarible is a multichain NFT marketplace and aggregator providing a unified REST API for searching, trading, and managing NFTs across Ethereum, Polygon, Arbitrum, Flow, IMX, and Chiliz. The platform operates as both a consumer marketplace and a B2B API infrastructure layer powering over 5,000 NFT applications.

The Rarible API exposes endpoints for querying NFT items and collections, order books (aggregated from OpenSea, Rarible, and other venues), ownership records, trading activities, floor prices, volume rankings, and analytics. An integrated TypeScript SDK handles wallet connections, NFT minting, listing, bidding, and purchases.

**Base URL:** `https://api.rarible.org/v0.1`

**Authentication:** `X-API-KEY` request header

**Documentation:** https://docs.rarible.org

**Pricing:** https://rarible.org/pricing

This repository contains the APIs.json 0.19 profile for Rarible maintained by [API Evangelist](https://apievangelist.com).
